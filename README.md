README.md

**-Aplikasi pemesanan tiket wisata lokal-**
Melakukan pemesanan tiket melalui website dan mencetaknya (opsional)

**Fitur yang tersedia :**
- Daftar
- Login
- Halaman Utama
- Destinasi
- Buat Pesanan
- Cek Pemesanan
- Dashboard User
- Riwayat Pemesanan User
- Dashboard Admin
- Backup Database
- Kelola Destinasi
- Pemesanan Semua User
- Menampilkan Daftar User
- Menampilkan Laporan Pemesanan

Pengguna
- Admin
- Pelanggan
- Guest

Penjelasan :
- Guest dapat membeli pesan tanpa harus mendaftar/membuat akun terlebih dahulu,setelah melakukan pembelian maka akan diberikan kode tiket yang harus disimpan agar dapat dilacak
- Saat guest sudah melakukan pemesanan,bisa melakukan pendaftaran untuk menyimpan Riwayat pembelian yang tersimpan dalam akun yang kita daftarkan
- Admin mengelola data destinasi, user, konfirmasi dll.
